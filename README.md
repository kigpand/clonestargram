## instragram clone coding site

### 현재 서버 이슈로 인해 배포 중단 중입니다.

### 기능 

* login, logout, 회원가입 가능
* 무한 스크롤링 구현
* 게시글 등록, 해시태그, 팔로워, 팔로잉 가능
* 데스크톱, 모바일 양쪽 모두 가능
___
### Tech
>* Next.js + Typescript 기반
>* custom hook - useInput(input 기능), useFetch(server 통신용), useWindowHook(window size 반응)
>* 상태관리 zustand 활용
>* css 전처리기 사용(scss)
